# projetoEAD Microsserviços

## Descrição
Projeto de microsserviços simulando uma plataforma de pedidos utilizando Docker e Docker Compose.

## Como rodar o projeto

1. Clone o repositório  
2. Execute:

3. Acesse:  
Acesse via porta 3000 no Codespaces

## Arquitetura
- Serviço de Pedidos (Node.js)
- Containerizado com Docker
- Orquestrado com Docker Compose

## Proposta Kubernetes
O sistema pode ser migrado para Kubernetes utilizando:
- Deployment
- Service
- ConfigMap
- HPA para escalabilidade

## CI/CD
Sugestão de uso com GitHub Actions para:
- Build automático
- Testes
- Deploy

## Observabilidade
Sugestão:
- Logs com console
- Métricas com Prometheus
- Tracing com OpenTelemetry
